# 4-bit Comparator (Verilog)

## Overview
This project implements a 4-bit comparator using Verilog to compare two binary inputs.

## Features
- Compares two 4-bit inputs (A and B)
- Outputs: A_greater, A_equal, A_less
- Implemented using combinational logic

## Technologies
- Verilog
- Digital Logic Design
